# Keyboard Shortcuts

## Global Shortcuts

### Command Palette
- **Mac**: `⌘ + K`
- **Windows/Linux**: `Ctrl + K`

Opens the command palette for quick navigation and actions.

## Command Palette Shortcuts

When the command palette is open:

- `↑` / `↓` - Navigate between items
- `Enter` - Select highlighted item
- `Esc` - Close the palette

## Future Shortcuts (Planned)

### Navigation
- `⌘ + 1` - Dashboard

### Actions
- `⌘ + N` - New Work Order
- `⌘ + Shift + N` - New Asset
- `⌘ + /` - Toggle sidebar

### Work Order Details
- `⌘ + S` - Save changes
- `⌘ + Enter` - Complete work order
- `Esc` - Close drawer/dialog

## Tips

- Keyboard shortcuts work from anywhere in the app
- You don't need to be focused on a specific element
- If a shortcut doesn't work, click outside any input field first
- Shortcuts are case-insensitive

## Customization

Currently, keyboard shortcuts are fixed. Custom shortcuts may be added in a future update.
