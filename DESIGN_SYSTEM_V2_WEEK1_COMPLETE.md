## üí° How to Use the New Sections

### For New Features:

1. **Check Typography Hierarchy**
   - Page title? Use text-2xl font-bold
   - Section title? Use text-lg font-semibold
   - Body text? Use text-sm font-normal

2. **Check Spacing System**
   - Between sections? Use space-y-8
   - Between cards? Use gap-6
   - Between form fields? Use space-y-4

3. **Add Loading States**
   - Button action? Add spinner + "Saving..."
   - Page load? Use skeleton loaders
   - File upload? Use progress bar

4. **Handle Errors**
   - Form error? Show inline + red border
   - API error? Show alert with details
   - No data? Show empty state with action

5. **Build Tables**
   - Use column width guidelines
   - Add loading state (skeleton rows)
   - Add empty state (no data)
   - Add pagination (>25 rows)
