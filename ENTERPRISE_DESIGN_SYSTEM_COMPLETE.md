## Design Principles

### Border-Based Design
- All components use borders instead of shadows
- Consistent border color: `border-gray-200`
- Clean, flat aesthetic

### Purple Primary Color
- Primary actions: `bg-purple-600` / `hover:bg-purple-700`
- Active states: `bg-purple-50` / `text-purple-900`
- Focus rings: `ring-purple-600`

### Consistent Sizing
- Input height: `h-9` (36px)
- Button padding: `px-4 py-2`
- Icon sizes: `w-3.5 h-3.5` (content), `w-4 h-4` (navigation)
- Border radius: `rounded-md` (inputs/buttons), `rounded-lg` (panels)

### Typography Standards
- Page titles: `text-2xl font-semibold`
- Section headers: `text-xs font-semibold uppercase tracking-wide`
- Body text: `text-sm text-gray-900`
- Labels: `text-xs font-medium text-gray-700`
- Helper text: `text-xs text-gray-500`

### Spacing Standards
- Component padding: `p-3` or `p-4`
- Section spacing: `space-y-4` or `space-y-6`
- Element gaps: `gap-2` (tight), `gap-4` (comfortable)
- Page padding: `p-4` (mobile), `p-6` (desktop)
